# CompSciRessources
Static website for first semester students earning an Associates Degree in Computer Science 

### Vigtige Ressourcer for 1. Semester Datamatikere

Her er en liste med ressourcer, der kan være eller vil blive relevante. Det er bl.a. værktøjer, software, og platforme, som I kan få adgang til gratis gennem jeres studiemail eller verificere med jeres ***studiekort*** derfor er det også vigtigt, at I får anskaffet jeres eget studiekort.

#### **IDEs (Integrated Development Environments)**

En IDE er en softwareapplikation, der giver udviklere mulighed for at skrive, teste og debugge kode effektivt. Her er nogle IDE'er, der er nyttige for studerende.

- [**Visual Studio Code (VS Code)**](https://code.visualstudio.com/)  
   En gratis kodeeditor fra Microsoft, som understøtter mange programmeringssprog, herunder C#, HTML og CSS. Det er letvægts, har mange udvidelser og kan nemt integreres med Git og andre værktøjer.  
   *Hvorfor det er nyttigt?*:  
   Fleksibel, let at bruge og kan tilpasses med udvidelser til debugging, GitHub-integration, og meget mere.

- [**Visual Studio (Community Edition)**](https://visualstudio.microsoft.com/vs/community/)  
   Et fuldt udbygget IDE fra Microsoft, specielt godt til C#-udvikling. Det er gratis for studerende og kommer med kraftige debugging og kodeanalyseværktøjer.  
   *Hvorfor det er nyttigt?*:  
   Ideelt til C#-udvikling, Windows-apps og større projekter med dybere integrationsmuligheder end VS Code.

- [**Visual Studio 2022 (Enterprise Edition)**](https://visualstudio.microsoft.com/vs/enterprise/)  
   En avanceret version af Visual Studio med flere professionelle funktioner såsom avancerede testværktøjer, arkitekturovervågning, og teamudviklingssupport. Det er primært designet til større projekter og teams.  
   *Hvorfor det er nyttigt?*:  
   Tilbyder funktioner for større virksomheder eller projekter med komplekse behov, såsom dybere integration af cloudtjenester og enterprise-skala udviklingsværktøjer.

- **Sammenligning**  
   Visual Studio Code er ideel til studerende og mindre projekter, hvor fleksibilitet og letvægtsstruktur er i fokus. Visual Studio Community er velegnet til mellemstore projekter, især når man arbejder med C# og Windows-applikationer. Visual Studio Enterprise er til store projekter med komplekse krav og har de mest avancerede funktioner til test, arkitektur og samarbejde i store udviklingsteams.
   I har adgang til Enterprise Edition, så brug den. Visual Studio Code bliver ofte forkortet som 'VSCODE' og Visual Studio 2022 (uanset edition) som VS2022.

#### **Diagramværktøjer (Flowcharts og Diagrammer)**

Diagrammer er vigtige til at visualisere systemarkitektur, datamodellering og processtyring.

- [**Draw.io**](https://app.diagrams.net/)  
   Et gratis web-baseret værktøj til at oprette flowcharts, ER-diagrammer, UML-diagrammer og meget mere. Det integreres med Google Drive, hvilket gør det nemt at samarbejde.  
   *Hvorfor det er nyttigt?*:  
   Gratis, intuitivt og nemt at dele og samarbejde på projekter.

- [**Microsoft Visio**](https://www.microsoft.com/en-us/microsoft-365/visio/flowchart-software)  
   Et betalt værktøj til mere komplekse diagrammer, der tilbyder professionelle templates og integration med andre Microsoft Office-produkter.  
   *Hvorfor det er nyttigt?*:  
   Mere avanceret og detaljeret end Draw.io, godt til større eller kommercielle projekter.

- [**Mermaidchart**](https://www.mermaidchart.com/)  
   Et onlineværktøj, hvor man "skriver" sine diagrammer i stedet for at tegne dem ved hjælp af en Markdown-lignende syntaks. Det understøtter flowcharts, sekvensdiagrammer, ER-diagrammer og meget mere.  
   *Hvorfor det er nyttigt?*:  
   Perfekt for dem, der foretrækker tekstbaseret diagramoprettelse, hvilket kan være hurtigere og lettere at vedligeholde i projekter, der kræver hyppige opdateringer af diagrammer.

- **Sammenligning**  
   Draw.io er gratis og brugervenligt til hurtigt at lave diagrammer, mens Visio tilbyder flere professionelle funktioner til mere komplekse behov. Mermaidcharts er ideel for dem, der ønsker at skabe diagrammer ved hjælp af tekstbaserede input, hvilket gør det velegnet til udviklere, der arbejder med kildekodekontrolsystemer. Visio er bedst egnet til avancerede brugere eller større virksomhedsapplikationer, mens Mermaidcharts giver en minimalistisk tilgang til diagrammering med mulighed for hurtig redigering.

#### **Databaser**

At lære SQL og databasesystemer er en vigtig del af datalogi. Her er et par værktøjer og læringsressourcer til at komme i gang.

- [**SQL Server Management Studio (SSMS)**](https://learn.microsoft.com/en-us/sql/ssms/download-sql-server-management-studio-ssms?view=sql-server-ver15)  
   Et værktøj fra Microsoft til at administrere SQL-databaser, hvor studerende kan oprette, forespørge og administrere databaser.  
   *Hvorfor det er nyttigt?*:  
   Vigtigt til at administrere og øve SQL på en virkelig database, da det understøtter alle typer SQL-serveropgaver.

- [**SQLZoo**](https://sqlzoo.net/)  
   En gratis platform, hvor studerende kan lære SQL ved at øve på live-databaser. Den giver øjeblikkelig feedback og hjælper med at udvikle SQL-færdigheder.  
   *Hvorfor det er nyttigt?*:  
   Hands-on erfaring er essentiel for at lære SQL, og SQLZoo tilbyder interaktive udfordringer for at forbedre evnerne til at skrive forespørgsler.

- **Bemærk**  
   SQL er et *sprog* og der er mange dialekter. Fx så bruger Microsoft T-SQL i bl.a. MSSQL og SSMS, hvor Oracle bruger PL/SQL eller MySQL. Forskellene er umiddelbart små men de er oftest inkompatible.

#### **Projektstyring og Versionskontrol**

At lære versionsstyring med Git er en essentiel færdighed for softwareudviklere, især når man arbejder på større projekter. Bemærk at Git og GitHub har samme relation som C# (værktøj) og Visual Studio 2022 (platform til at anvende værktøjet).

- [**GitHub**](https://github.com/)  
   En platform for versionsstyring, der bruger Git. Det hjælper med at spore kodeændringer, arbejde sammen med andre på projekter og gemme kode i repositories.  
   *Hvorfor det er nyttigt?*:  
   Introducerer bedste praksis i softwareudvikling og giver studerende mulighed for at opbygge deres kodeporteføljer.

- [**Codeberg**](https://codeberg.org/)  
   En gratis og open-source platform til versionsstyring, der er designet til dem, der ønsker en 100% fri og non-profit version af Git-tjenester. Codeberg drives af frivillige og understøtter samarbejde om open-source projekter.  
   *Hvorfor det er nyttigt?*:  
   Perfekt for studerende, der ønsker at arbejde med eller bidrage til open-source projekter uden nogen kommercielle interesser.

- **Sammenligning**  
   GitHub er den mest udbredte platform til versionsstyring og samarbejde, med et stort fællesskab og integration til mange udviklingsværktøjer. Codeberg tilbyder en mere uafhængig, non-profit tilgang til Git-tjenester, hvilket kan være attraktivt for dem, der foretrækker open-source løsninger uden kommercielle bindinger. GitHub har flere integrationer med værktøjer som CI/CD, men Codeberg er en stærk kandidat for dem, der vægter frihed og uafhængighed højt.
   I vil nok blive tvunget til GitHub og det er også bedst integreret med VSCODE og VS2022

#### **Læringsplatforme**

Disse platforme tilbyder strukturerede kurser, der passer til det materiale, studerende vil arbejde med på første semester.

- [**LinkedIn Learning**](https://www.linkedin.com/learning/)  
   I har adgang til dette gennem jeres studiemail. LinkedIn Learning er en omfattende platform, hvor bl.a. Microsoft ofte tilføjer nye kurser og opdaterer eksisterende. Når et kursus gennemføres, kan man få et certifikat, som kan tilføjes til sin LinkedIn-profil for at fremvise ens kompetencer.  
   *Hvorfor det er nyttigt?*:  
   Struktureret læring og en god måde at få præsenteret materialet på en anden måde. Kurserne kan nogle gange føles lidt overfladiske eller for "amerikanske," hvor man bliver spoon-fed information, men det giver mulighed for at kode parallelt. Her er nogle relevante kurser:  
   
   - [**LinkedIn Learning - C# Developer Path**](https://www.linkedin.com/learning/paths/getting-started-as-a-c-sharp-developer?u=37487420)  
     Dette kursus dækker alt fra grundlæggende syntaks til objektorienteret programmering i C#. Perfekt til begyndere og understøtter det, I lærer i undervisningen.  
   
   - [**Become a Programmer: Foundations**](https://www.linkedin.com/learning/paths/become-a-programmer-foundations?u=37487420)  
     Dækker grundlæggende programmeringsprincipper, men tager også kort fat i andre programmeringssprog end C#. Vær dog opmærksom på, at det ikke er lige så dybdegående som jeres lektioner i C#.

- [**Microsoft Learn**](https://learn.microsoft.com/en-us/training/)  
   Microsofts egen læringsplatform, som dækker teknologier som SQL, C#, Azure og meget mere. Platformen tilbyder interaktive moduler og quizzes, som hjælper jer med at følge jeres fremskridt og opbygge forståelse trin for trin. Indholdet er dog skrevet af dem, der har skrevet bl.a. C# så de går ofte teknisk langt i dybden, hvilket kan være forvirrende, når man bare vil se nogle simple eksempler.  
   *Hvorfor det er nyttigt?*:  
   Officiel og struktureret læring med moduler, der er målrettet de teknologier, I kommer til at arbejde med både i studiet og fremtidige projekter.

- **Sammenligning**  
   Selvom både LinkedIn Learning og Microsoft Learn tilbyder stærke læringsplatforme, bør disse kurser ses som supplementer til jeres primære studier. De giver mulighed for at uddybe det, I lærer i undervisningen, og kan være en god måde at få alternative forklaringer på svære emner. Studiet og undervisningen er dog det primære, da det er mere tilpasset pensum og giver jer den dybdegående viden, der kræves til eksamener og fremtidige projekter. Brug derfor disse platforme til at understøtte jeres læring, men fokuser først og fremmest på det, I lærer i timerne.

#### **Cloud-Tjenester og Hosting**

At forstå cloud-teknologi og hosting er vigtigt for moderne udvikling.

- [**Azure for Students**](https://azure.microsoft.com/en-us/free/students/)  
   En gratis service, hvor studerende får credits til at eksperimentere med hosting, cloud computing, databaser og mere.  

   *Hvorfor det er nyttigt?*:  
   Giver mulighed for at arbejde med cloud-tjenester gratis og lære om hosting og databaser i cloud-miljøet.  

   *Bemærk:*  
    Azure kan virke som et uoverskueligt monster, fordi det er et uoverskueligt monster. Der er alt fra simpel hosting til brugere, der kører flere virtual machines med multimodale LLM'er. Dyb vejrtrækning og fokuser på det i vil anvende Azure til. 
    Gem jeres credits til senere semestre og sørg for at oprette Cost and Billing Plans, der er 0 dollars eller Free-tier, for Azure kan suge ens pengepung hurtigere tom end.. well.. brug ChatGPT eller jeres fantasi til at afslutte den sætning.

#### **Support og Hjælp**

At have adgang til hjælp og support er afgørende, især når man støder på problemer med koden.

- [**GitHub Student Developer Pack**](https://education.github.com/pack)  
   En samling af gratis værktøjer og tjenester for studerende, der inkluderer gratis private repositories og adgang til premium-udviklingsværktøjer.  
   *Hvorfor det er nyttigt?*:  
   Perfekt til at eksperimentere med nye teknologier, udvikle projekter og få adgang til værktøjer til udvikling, design og opbevaring.

- [**Stack Overflow**](https://stackoverflow.com/)  
   En stor online platform for programmører, hvor studerende kan stille spørgsmål, søge efter svar og få hjælp til debugging fra fællesskabet.  
   *Hvorfor det er nyttigt?*:  
   Hjælper studerende med at lære af rigtige kodeproblemer og tilbyder support fra et stort udviklerfællesskab.
